# Editor badge

Use the **editor** badge for stories with a live editor. The badge displays ✏️ with an "Editor" tooltip.

## Usage

Add the `editor` tag to your story.

```tsx
export const YourStory = {
	tags: ['editor'],
	render: () => <YourComponent />
}
```
