# Var badge

Use the **var** badge for stories that describe values and variables. The badge displays **var** with a "Variables" tooltip. This badge is hidden in MDX docs.

## Usage

Add the `var` tag to your meta (default export). No need to add it to each story.

```tsx
export default {
	tags: ['var'],
	// ...
} satisfies Meta

export const YourStory = {
	render: () => <YourComponent />
}
```

Stories in a file whose meta is tagged with `var` will show the **var** badge in the sidebar.
