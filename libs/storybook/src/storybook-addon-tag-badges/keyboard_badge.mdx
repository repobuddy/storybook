# Keyboard badge

Use the **keyboard** badge for stories with keyboard interactions. The badge displays ⌨️ with a "Keyboard Interaction" tooltip.

## Usage

Add the `keyboard` tag to your story.

```tsx
export const YourStory = {
	tags: ['keyboard'],
	render: () => <YourComponent />
}
```
